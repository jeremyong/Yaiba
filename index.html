Yaiba by jeremyong

Yet Another Implementation of Buchberger's Algorithm. Haddock generated documentation.

Attempts to use concurrency libraries in Haskell to implement Buchberger's Algorithm for computing Groebner bases of multivariate polynomial ideals.

Dependencies

Build-Depends: base == 4.*, dph-base == 0.4.*, repa == 1.1.*
